# 🔐 Login Page

Página de login moderna desenvolvida com HTML, CSS e JavaScript puro.

## 🎨 Visual

![Login Page](img/preview.png)

## Tecnologias
- HTML5
- CSS3 (Glassmorphism, Flexbox)
- JavaScript (DOM, eventos, validação)

## Funcionalidades
- Layout moderno com efeito glassmorphism
- Validação de login com JavaScript
- Ícones com Boxicons
- Google Fonts
- Responsivo

## Como usar
1. Clone o repositório
2. Abra o `index.html` no navegador
